笔记

  1. 文本格式化标签

加粗 strong></strong>（推荐）和<b></b>
倾斜<em></em>（推荐）和<i></i>
删除线<del></del>（推荐）和<s></s>
下划线<ins></ins>（推荐）和<u></u>

  1. div和span

    大盒子，独占一行 小盒子

  2. a标签
    href必须以http://开头
    媒体元素都可以作为超链接对象

<a href="practice.html" target="framename" ><img src="img.jpg" alt="neibulianjie" title="test" width="150" border="10" /></a>

空链接

<a href="#"></a>

锚点链接

<a href="#123"></a>
<h1 id="#123"></h1> （跳转对象）

  1. 表格

    表头单元格里面的内容加粗居中显示 cellpadding 规定单元边沿与其内容之间的空白，默认1像素 cellspacing 规定单元格之间的空白，默认2像素 跨行合并：rowspan=“合并单元格的个数” 跨列合并：colspan="合并单元格的个数"

  2. 列表
    <dl></dl>
    自定义列表
    里面只能包含dt和dd，dt和dd里面可以放任何标签

      
    <dl>
<dt>关注我们</dt>
<dd>新浪微博</dd>
<dd>联系我们</dd>
</dl>

  3. 表单域（from）

input输入表单元素

  • type属性

  • name属性
    定义 input 元素的名称

  • value属性
    规定 input 元素的值（可提供文本输入框默认值显示）

  • checked属性(页面打开默认选中)
    单选按钮和复选框可以设置checked 属性

  • maxlength属性较少使用 

输入最大长度
  • label标签
label标签用于绑定一个表单元素
label标签的 for属性 应当与相关元素的id 属性相同

<label for="sex"> 男 </lable>
<input type="radio" name="sex" id="sex" />

select表单元素

<select>
<option selected="selected">选项1</option>
<option>选项2</option>
<option>选项3</option>
</select>

  • 在option中定义 selected=”selected” 时，当前项即为默认选中项。

杂记

  1. head标签
    <head></head>
    下面这些标签可用在 head 部分：
    <base>, <link>, <meta>, <script>, <style>，<title>
    <title>
    是 head 部分中唯一必需的标签。

  2. 空标签
    没有元素内容的标签，例如<img>,<br>等等这种单个标签

  3. 自适应
    <meta name="viewport" content="width=device-width,initial-scale=1.0" >

有关自适应详细信息

  1. a标签的target属性
    <a target = ""></a>
    指定在何处打开超链接。
    _blank：在新窗口中打开
    _parent：在当前的父窗口中打开，如果不存在父窗口，此选项的行为方式与 _self 等同
    _self：当前窗口打开（默认）
    _top：在整个窗口中打开
    framename：在指定的框架中打开
  1. 基于百度官方的SEO
    善用Meta description

<meta name="keywords" content="">
<meta name="description" content="">
<meta name="author" content="">

  1. meta中的http-equiv标签重定向
    <meta http-equiv="refresh" content="5; http://baidu.com">

  2. style中media属性
    media 属性可用来表明文档在什么情况下应该使用该元素中定义的样式。
    例：

    <style type="text/css" media="print">
h1 {color:#000000;}
p {color:green;}
body {background-color:#FFFFFF;}
</style>

    打印或预览这个页面，”print”样式包含了绿色文本和白色背景
    详细信息

  1. 利用link中的rel属性为网页设置favicon
    <link rel="icon" type="image/x-icon" herf="" />
